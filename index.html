Welcome to S.A.B.A.S
(Smart Automated Batch for Attendance System)

This project includes ESP32 Cam Module, ESP32 Dev Module and an RFID Reader Module.
Uses :

1.) ESP32 Cam Module - The Cam Module is being used for the 3D Body Modelling and student recognition. It is used for the double verification of the student and a part of the security system.


2.) ESP32 Dev Module - Dev Module is used for controlling incoming and outgoing signals and also it is the cpu of our system which controls all the components in our system.


3.) RFID Reader Module - RFID Reader is used for the student identification and attendance marking. It is used for the single verification of the student. It tells the Dev Module about the student UID using which the dev Module can understand which is this student.
